# Book Listing

This is the task required by Leap13 Hiring.

## Features

- **Add Books**: User can add his favorite books through a form including book name, author, release date and cover image (file type input field).
- **Remove Books**: Each book item in the list has a "Remove From List" button that allows users to remove the book from their list.
- **Search Books**: User can search for books by name or author using separate search bars.
- **Sort Books**: User can sort books by name, author or release date.
